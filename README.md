
Travel Tracker is a web-based application designed to collect and manage travel behavior data for research and planning purposes. The system allows users to record detailed trip information such as origin and destination locations, travel time, travel mode, distance, trip purpose, cost, and number of companions. The application stores the collected data in a structured database and processes it to generate useful insights like trip frequency, mode usage patterns, and weekly travel trends.

The backend of the application is developed using Flask, while data storage is handled through SQLite. The application also incorporates basic data processing techniques to analyze the collected travel information and support transportation research.

This system can assist researchers and planners in understanding travel patterns and making data-driven decisions for transportation planning and policy development.
